<h1 align="center">Project Title: Meta Photos</h1>

<p>Meta Photos is a web application designed as a clone of Google Photos, serving as a user-friendly platform for photo uploading and management. It allows users to register, log in, and securely upload photos to their personal gallery. The platform ensures each user's photos are stored and displayed based on their unique login credentials.</p>
